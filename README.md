# xcel

A simulation environment for the FS Driverless competition. Primarily built on JAX & MuJoCo.

Thanks to [AMZ's simulator](https://github.com/AMZ-Racing/fssim) for the track and cone models!

Primarily designed to work with [Drivebrain](https://github.com/hytech-racing/drivebrain) from HyTech Racing.
